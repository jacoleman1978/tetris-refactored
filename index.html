About the game

×

Rotating the shapes clockwise or counterclockwise:

There are 7 different possible shapes that can be made using four squares. Most of these shapes can be rotated in up to four different orientations using the 'D' and 'F' keys on the keyboard or the curved arrow buttons on the side of the game grid area.

Moving the shapes left or right:

The shapes will appear at the top of the game grid area and fall. You can move the shape left or right using the arrow keys on the keyboard or the arrow bottons on the side of the game grid area.
